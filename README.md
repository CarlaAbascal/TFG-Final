# TFG-Final





Aquest repositori conté el codi desenvolupat per al Treball de Fi de Grau, organitzat en diferents carpetes segons les funcionalitats implementades durant el projecte.



\## Estructura del repositori



\* `Aplicació final/`: conté l’aplicació principal desenvolupada en C# Windows Forms, on s’integren les diferents funcionalitats del projecte.

\* `Reconeixement de Veu/`: conté el projecte relacionat amb el reconeixement de veu i la interpretació de comandes vocals.

\* `Streaming de Video/`: conté el projecte encarregat de la transmissió i visualització de vídeo en temps real.

\* `Reconeixement de Gestos/`: conté el sistema de reconeixement de gestos mitjançant visió per computador.

\* `Reconeixement d'Objectes/`: conté el sistema de detecció d’objectes utilitzant tècniques de visió per computador.



Cada carpeta inclou el seu propi fitxer `README.md`, on s’expliquen els requisits necessaris, les dependències i els passos per executar el projecte de manera independent.
